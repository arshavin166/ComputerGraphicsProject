# Komande

`F1` - on/off Imgui

`G`  - on/off Gamma

`L`  - switch between Phong/Blinn-Phond model light

`N` - lower Exposure

`M` - higher Exposure

`H` - off/on HDR

`B` - off/on Bloom

`X` - off/on Grayscale
