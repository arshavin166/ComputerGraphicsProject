Projekat iz Računarske Grafike.

---------------------
# Komande

`F1` - on/off imgui

`G`  - on/off grayscale

`L`  - switch between Phong/Blinn-Phond model light 

`J` - lower exposure

`K` - higher exposure

`H` - off/on HDR 

`B` - off/on bloom
